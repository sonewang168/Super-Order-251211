## 🚀 快速開始

### 1️⃣ 開啟 APP
直接雙擊 `index.html` 即可使用！

### 2️⃣ 設定 API Key
1. 進入設定頁 ⚙️
2. 輸入 [Gemini API Key](https://aistudio.google.com/apikey)
3. 點擊「💾 儲存」

### 3️⃣ 開始生成
1. 輸入主題描述
2. 選擇風格
3. 🚀 開始生成！

## ☁️ 雲端功能設定

如需使用 Google 相簿、插畫圖鑑、LINE 通知功能：

1. 部署 `Code.gs` 到 [Google Apps Script](https://script.google.com)
2. 詳見 [docs/SETUP.md](docs/SETUP.md)
